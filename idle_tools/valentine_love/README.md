# 520 浪漫空间 (valentine_love)

**5·20 · 我爱你** — 打开即自动播放的五幕浪漫动画，可打包为单文件 exe。

## 运行

| 方式 | 说明 |
|------|------|
| **`dist\520浪漫盒.exe`** | 只发这一个文件，对方双击即看（需 Edge/Chrome） |
| **`520浪漫盒.bat`** | 本地预览 |

## 自动分镜（无需对方操作）

1. **520 开场** — 巨幕 + 烟花  
2. **金句** — 情话逐句浮现  
3. **心核** — 520 / 我爱你 + 漂浮文字  
4. **情书** — 全屏打字机  
5. **终章** — 520 快乐  

点击屏幕或空格可跳到下一幕。

## 发送前定制（你改，对方不用填）

编辑 **`js/config.js`**：

- `toName`：称呼（如 `"宝贝"`，留空则用「你」）
- `fromName`：落款
- `vowLines`：第二幕金句
- `letterLines`：情书正文

改完后执行 **`build_exe.bat`**，再发送 `dist\520浪漫盒.exe`。

## 打包

```text
idle_tools\valentine_love\build_exe.bat
```

输出：`dist\520浪漫盒.exe`
